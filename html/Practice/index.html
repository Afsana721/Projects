HTML Attributes

HTML attributes provide additional information about their HTML elements

What are HIML Attributes

  • All HTML elements can have attributes
  • Attributes provide additional information about their parent elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name ="value"

The href Attribute
