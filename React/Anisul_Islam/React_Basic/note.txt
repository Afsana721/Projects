What is React?
    - React is flexible, efficient , open-source javascript library. 
    - Reusable component, Load fast, External Plugin, Around 45% of world's website use React
    - major brands like Facebook, Instagram, Yahoo, Netflix, Airbnb, dropbox using react.

    -It wia developed by Facebook back in 2013 fo rbuinding user interfaces (front end.).
    - Facebook software Engineer, Jordan Walke, created it.
Why React?
    - It allows us to create a complex UI by making components (samll & isolated pieces of code)
    - Components are reusable.
Example: 
    Message.js file has message components

    import React from 'react';
    export default function Message(){
        return(
            <h1 style={{backgroundColor: "red", textAlign: "center"}}>
            Welcome to React
            </h1>
        )
    }
So, we can use this message function/ method anywhere again & anagin

    - < Message />
    - < Message />


What is JSX?
    - JSX stands for JavaScript XML.
    - JSX allows us sto write and add HTML in React.
    - React module has a compiler name babel - it convert HTML code to JavaScript code.
