Sample 3: pass function inside component

const Home = () => {
// create a function to click the button
const handleClick = () => {
    console.log('hello, ninjas')
}

const handleClickAgain = (name) => {
    console.log("Hello " +name)
}
    return ( 
        <div className="home">
            <h2>HomePage</h2>
            <button onClick={handleClick}>Click me</button>
            {/* not work like : <button onClick={handleClickAgain(name)}>Click me again</button> */}
        
            <button onClick={() => {
                handleClickAgain("Asef")
            }}>Click me again</button>    
        </div>
     );
}
 
export default Home;

//explanation:
{/* not work like : <button onClick={handleClickAgain(name)}>Click me again</button> */}
        
            <button onClick={() => {
                handleClickAgain("Asef")
            }}>Click me again</button>    

we can't add the handleClickAgain(Masud) - function with passing parameter because that
time, the function will autmatically invoke, running autmatically. 
So, we create a anonymous function and invoke the anonymous function to execute handleClickAgain function
with the parameter. 
Now the handleClickAgain function is not invoke because it is wrapped inside the anonymous
function. So when click the button then first anonymous function will be fired then invoke the 
handleClickAgain function.
